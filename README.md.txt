# Iris Flower Classification 

This project uses machine learning to classify iris flowers into different species using the Iris dataset.

## Features
- Classification using ML algorithms
- Data preprocessing and training
- Accuracy evaluation

## Technologies Used
- Python
- Scikit-learn
- Pandas
- NumPy

## Output
Predicts iris flower species based on input features.

## Dataset
Iris dataset
